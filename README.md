# Linux Administration Scripts

This repository contains **real-world Bash scripts** used in Linux production
environments for user management, monitoring, and backups.

## Scripts Included
- user_add.sh – User creation with expiry
- disk_usage_alert.sh – Disk usage monitoring
- service_monitor.sh – Critical service health check
- backup_rsync.sh – Automated backups using rsync

## Usage
```bash
chmod +x *.sh
./script_name.sh
